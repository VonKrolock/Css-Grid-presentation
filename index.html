What is CSS Grid

The CSS Grid is a two-dimensional layout system that can handle both rows and columns.

Why CSS Grid

  • CSS Grid make it easier to design web pages without having to use floats and positioning
  • CSS Grid is supported in the latest versions of all major browsers and can be safely used with a fallback

Grid vs Flexbox

Unlike Flexbox, CSS Grid is a two-dimensional layout system that can handle both rows and columns.

CSS Grid Container

  • The grid container is the parent of all grid items.
  • It is declared with the "display: grid" or "display: inline-grid" rule
    • grid – generates a block-level grid
    • inline-grid – generates an inline-level grid

CSS Grid Item

The items that are direct children of the grid container. These are the elements that will be placed on the grid.

CSS Grid Columns and Rows

  • The vertical lines of grid items are called columns
  • The horizontal lines of grid items are called rows

CSS Grid Lines

The lines that divide the columns and rows of the grid

  • The lines between columns are called column lines.
  • The lines between rows are called row lines.

CSS Grid Gaps

  • The spaces between each column/row are called gaps
  • You can adjust the gap size by using one of the following properties:
    • grid-column-gap
    • grid-row-gap
    • grid-gap

grid-template-columns

  • The grid-template-columns property defines the number of columns in your grid layout, and it can define the width of each column
    • grid-template-columns: auto auto auto auto
    • grid-template-columns: 80px 200px auto 40px
  • If you have more than 4 items in a 4 columns grid, the grid will automatically add a new row

grid-template-rows

  • The grid-template-rows property defines the height of each row
  • The number of rows is determined by the number of values defined in the space-separated list
    • grid-template-rows: 80px 100px - two rows
    • grid-template-rows: 80px 100px 150px - three rows

grid-column

  • The grid-column property is a shorthand property for the grid-column-start and the grid-column-end properties
  • The grid-column property defines on which column to place an item

To place an item, you can refer to line numbers, or use the keyword "span" to define how many columns the item will span

  • "grid-column: 1 / span 3" - start on column 1 and span 3 columns
  • "grid-column: 1 / 5" - start on column 1 and end before column 5

grid-row

  • The grid-row property is a shorthand property for the grid-row-start and the grid-row-end properties
  • The grid-row property defines on which row to place an item

To place an item, you can refer to line numbers, or use the keyword "span" to define how many rows the item will span

  • "grid-row: 1 / span 2" - start on row 1 and span 2 rows
  • "grid-row: 1 / span 4" - start on row-line 1 and end on row-line 4

grid-area

  • The grid-area property can be used as a shorthand property for the grid-row-start, grid-column-start, grid-row-end and the grid-column-end properties
    • "grid-area: 1 / 2 / 5 / 6" - start on row-line 1 and column-line 2, and end on row-line 5 and column line 6
  • The grid-area property can also be used to assign names to grid items

justify-content

  • The justify-content property is used to align the whole grid inside the container
  • The grid's total width has to be less than the container's width for the justify-content property to have any effect

align-content

  • The align-content property is used to vertically align the whole grid inside the container
  • The grid's total height has to be less than the container's height for the align-content property to have any effect

place-content

  • place-content sets both the align-content and justify-content properties in a single declaration
  • The first value sets align-content, the second value justify-content. If the second value is omitted, the first value is assigned to both properties
